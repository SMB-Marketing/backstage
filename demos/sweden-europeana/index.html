Sweden Europeana - Exhibit3 Staged Demo

The following interface is based on the initial Exhibit3-staged code base. Exhibit3-staged is a web service that that takes over heavy computations from the browser-based components to speed up the faceted navigation of large data sets. This work builds upon David Huynh's original "Backstage" work and reflects some of the capabilities we expect to achieve in the Exhibit 3 project.
